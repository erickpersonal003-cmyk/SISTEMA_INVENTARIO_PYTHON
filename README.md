## Descripción del proyecto
Este proyecto corresponde a la Evaluación en Contacto con el Docente de la asignatura Lógica de Programación. El sistema permite controlar productos mediante un inventario básico desarrollado en Python.

## Problema que resuelve
Muchos negocios pequeños o instituciones llevan el control de productos de forma manual, lo que puede generar errores, pérdida de información o desconocimiento del stock disponible. Este sistema busca apoyar el control básico de productos, cantidades y precios.

## Objetivo del sistema
Desarrollar un sistema básico de inventario en Python que permita registrar productos, consultar existencias, buscar productos, actualizar cantidades, registrar ventas y detectar productos con bajo stock.

## Funcionalidades principales
- Agregar productos.
- Mostrar inventario.
- Buscar productos.
- Actualizar cantidades.
- Registrar ventas o salidas.
- Mostrar productos con bajo stock.
- Salir del sistema.
